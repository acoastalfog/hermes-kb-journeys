# Hermes KB Journeys — retired

This repository is terminally deprecated. It must not be installed, deployed,
or used in production.

The installable plugin manifest, package metadata, generated runtime contracts,
implementation, tests, refresh tooling, and CI workflow were removed in the
terminal retirement commit. Their complete history remains available in Git.

NOC supports Codex and Claude as its named interactive harnesses. Durable
knowledge operations remain governed by `kb_engine_prod`; provider access is
supplied through harness-neutral native plugins, MCP servers, ai-pim-utils,
Source Access, and independently justified owned connectors.

Do not revive this repository as a bot, relay, scheduler, dashboard, connector
broker, or compatibility layer. Any future capability must be designed in its
current owning system without restoring Hermes-specific support.

The execution record and archive receipt are tracked in
[NOC Issue #874](https://github.com/acoastalfog/noc/issues/874).
